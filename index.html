Capacidad correcta

Mantenimiento

Cuatro cuidados basicos.

01

Limpia filtros

Revisalos cada mes si el equipo se usa con frecuencia o hay mucho polvo.

02

Evita obstrucciones

No cubras la salida de aire con muebles, cortinas o decoracion cercana.

03

Revisa drenaje

Goteos o malos olores pueden indicar suciedad, bloqueo o falta de mantenimiento.

04

Programa revision

Una revision tecnica periodica ayuda a detectar ruido, vibracion o baja eficiencia.

Preguntas frecuentes

Dudas comunes sobre aires acondicionados.

A que temperatura conviene usar el aire?

Una referencia comun de confort esta entre 23°C y 25°C, ajustando segun humedad, ropa, actividad y preferencia.

Cada cuanto limpiar el filtro?

En uso frecuente, una revision mensual es una buena base. Si hay polvo, mascotas o humedad alta, puede requerir mas frecuencia.

Que significa BTU?

Es una unidad usada para expresar capacidad de enfriamiento. A mayor area o calor interno, mayor capacidad requerida.

Por que el equipo gotea?

Puede ocurrir por drenaje obstruido, filtros sucios, instalacion con pendiente incorrecta o falta de mantenimiento.
